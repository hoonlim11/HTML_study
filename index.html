• 반드시 직접 써보면 좋은 핵심 태그

    • 구분 태그 설명
    문서 구조 <html>, <head>, <title>, <body> HTML 문서의 기본 골격을 정의
    텍스트 <h1>~<h6>, <p>, <br>, <hr>, <strong>, <em> 제목, 단락, 줄바꿈, 수평선, 강조 텍스트
    링크 & 이미지 <a>, <img> 하이퍼링크, 이미지 삽입
    목록 <ul>, <ol>, <li> 순서 있는/없는 목록 구성
    표 <table>, <tr>, <td>, <th> 표 구성, 행과 셀 정의
    입력 <input>, <label>, <textarea>, <select>, <option>, <button> 사용자와의 상호작용 요소 (폼 입력)
    시맨틱 구조 <header>, <footer>, <nav>, <main>, <section>, <article> 의미론적 웹 페이지 영역 구분
    기타 <abbr>, <address>, <mark>, <code> <pre> 특수한 목적의 표현 (약어, 주소, 강조 등)

  • 나중에 필요할 때 찾아보면 되는 태그

    • 구분 태그 설명
    미디어 처리 <canvas>, <video>, <audio> 그래픽, 동영상, 오디오 등 멀티미디어 콘텐츠 처리
    접힘/펼침 인터랙션 <details>, <summary> 클릭 시 내용을 접었다 펼칠 수 있는 UI 구성
    특수 상황 <iframe>, <map>, <progress> 외부 페이지 삽입, 이미지 맵, 진행률 등 특수 기능

  • HTML 문서 기본 구조 태그 설명 표 (링크 포함)

    • 태그 설명
    <!DOCTYPE html> HTML5 문서임을 브라우저에 선언함 (브라우저가 HTML5로 해석하도록 유도)
    <html lang="ko"> HTML 문서의 최상위 루트 요소이며, lang 속성으로 문서의 기본 언어를 지정 (예: 한글이면 ko)
    <head> 문서의 정보(meta 데이터, title, 외부 파일 링크 등)를 담는 영역으로 화면에는 보이지 않음
    <meta charset="UTF-8"> 문자의 인코딩을 UTF-8로 설정하여 한글, 특수문자 등이 깨지지 않도록 함
    <meta name="viewport"> 모바일/태블릿 등 다양한 화면 크기에 맞춰 콘텐츠가 잘 보이도록 설정 (반응형 웹의 필수 요소)
    <title> 브라우저 탭에 표시될 문서 제목이며, 검색엔진에서의 표시 제목에도 사용됨
    <body> 화면에 실제로 표시되는 본문 내용을 작성하는 영역 (텍스트, 이미지, 버튼 등 포함)

    The Main Languages of the Web

    HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page, and consists of a series of elements. HTML elements tell the browser how to display the content.

    CSS is a language that describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work, because it can control the layout of multiple web pages all at once.

    JavaScript is the programming language of HTML and the Web. JavaScript can change HTML content and attribute values. JavaScript can change CSS. JavaScript can hide and show HTML elements, and more.
